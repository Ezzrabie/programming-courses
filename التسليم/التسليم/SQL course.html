Level 1: What is SQL?

Discover what SQL is and why databases are essential. Get started with your first query: SELECT * FROM users;

Learn how to create tables and choose the right data types. Try: CREATE TABLE books (id INT, title VARCHAR(100));

Add new records to your tables using INSERT INTO. Example: INSERT INTO books VALUES (1, 'SQL Basics');

Level 4: Querying Data

Use SELECT, WHERE, ORDER BY and more to retrieve exactly what you need.

Level 5: Updating & Deleting

Change and remove data safely. Try: UPDATE users SET name = 'Amr' WHERE id = 1;

Level 6: Joins & Relationships

Connect data from multiple tables with INNER JOIN, LEFT JOIN, and more.
